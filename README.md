# 🧮 JavaScript Calculator

A simple and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations and demonstrates DOM manipulation, event handling, and JavaScript logic.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Parentheses support `(` `)`
- ⌫ Backspace functionality
- 🧹 Clear display (`C`)
- 🎨 Clean and responsive UI
- 🖱️ Button hover and click animations
- ⚠️ Basic error handling for invalid expressions

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
